# 项目介绍

用 Vant 1.x 和 Vue 2 搭建金融社区项目。功能有：知识库（在线预览文件，如word、pdf、富文本）、我的社区（像微博移动端界面一样）、金融市场（需要用图表、表格展示收益）、公告管理（富文本）。

包含如下功能：

- 基于 Vant 搭建
- 基于 vue-router 的单页面应用
- 组件按需引入
- 视图异步加载

> 技术栈：Vue 2、Vant 2、Vue Cli、Vue Router

## 开发命令

```bash
# 安装依赖
npm install

# 本地开发
npm run serve

# 生产环境构建
npm run build

# 代码格式校验
npm run lint
```
